# Shubham Mishra - Portfolio

A modern, professional portfolio website built with Next.js 15, TypeScript, and Tailwind CSS.

## Features

- **Modern Design**: Bluish-black and grey color scheme with smooth animations
- **Fully Responsive**: Optimized for all devices (mobile, tablet, desktop)
- **Fast Performance**: Built with Next.js 15 for optimal loading speed
- **Smooth Animations**: Powered by Framer Motion
- **SEO Optimized**: Proper meta tags and semantic HTML
- **Type-Safe**: Written in TypeScript

## Sections

- **Hero**: Animated introduction with profile photo and quick stats
- **Experience**: Timeline of professional work at Jio Platforms Limited
- **Projects**: Showcase of featured projects with GitHub links
- **Skills**: Interactive skill visualization with progress bars
- **Blog**: Developer stories and technical articles (coming soon)
- **Contact**: Contact form and social links

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **Icons**: React Icons
- **Deployment**: Vercel

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

### Build

```bash
npm run build
```

## Customization

Update the data files in `/data` directory:
- `personal.ts` - Personal info, bio, and stats
- `experience.ts` - Work experience
- `projects.ts` - Projects
- `skills.ts` - Technical skills
- `blog.ts` - Blog posts

Replace assets in `/public/assets/`:
- `profile.jpg` - Profile photo
- `Shubham_Mishra_Resume.pdf` - Resume
